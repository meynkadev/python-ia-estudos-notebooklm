# Projeto de Estudos com NotebookLM

## Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio da DIO com o objetivo de explorar o uso da Inteligência Artificial como ferramenta de aprendizado ativo.

A proposta central é utilizar IA para auxiliar nos estudos de Python, análise de dados, automação e organização do conhecimento técnico.

---

## Objetivos

- Aprender conceitos fundamentais de Python
- Explorar ferramentas de Inteligência Artificial aplicadas aos estudos
- Desenvolver pensamento lógico e analítico
- Organizar conteúdos técnicos de forma estruturada
- Construir uma base sólida para atuação na área de dados

---

## Tecnologias e Temas Estudados

- Python
- Inteligência Artificial
- Engenharia de Prompt
- NotebookLM
- Organização de Estudos
- Automação
- Análise de Dados

---

## Aprendizados

Durante o desenvolvimento deste projeto, foram explorados conceitos relacionados à utilização estratégica da Inteligência Artificial como apoio ao aprendizado técnico e organização de conhecimento.

Também foram desenvolvidas habilidades de documentação, estruturação de estudos e curadoria de informações.
