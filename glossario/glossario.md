# Glossário

## Python
Linguagem de programação amplamente utilizada em automação, análise de dados e Inteligência Artificial.

## IA (Inteligência Artificial)
Tecnologia capaz de simular comportamentos inteligentes através de algoritmos e modelos computacionais.

## Prompt
Instrução fornecida para uma Inteligência Artificial gerar respostas.

## Automação
Uso de tecnologia para executar tarefas automaticamente.
